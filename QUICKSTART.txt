📋 PREREQUISITES:
   - Node.js 18.17.0+
   - OpenAI API key (get from platform.openai.com/api-keys)

⚡ 3-STEP SETUP:

   1. Install dependencies:
      pnpm install

   2. Create .env.local file with your OpenAI API key:
      OPENAI_API_KEY=sk-your-key-here

   3. Run dev server:
      pnpm dev

🌐 OPEN: http://localhost:3000

🎯 FIRST TEST:
   1. Click "Start Teaching"
   2. Click "New Concept"
   3. Name: "Photosynthesis", Category: Biology
   4. Type: "Photosynthesis is how plants make food"
   5. Watch the AI learn in real-time!
